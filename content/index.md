---
title: Welcome to the Waterdeep Registry
---

This project is an Obsidian.md vault that includes a leaflet map of Waterdeep by Tych Maps, hosted via Quartz.

The goal of this project is to create a registry of buildings, sorted by block, neighborhood, district, and ward. This allows players, authors, and Dungeon Masters to easily locate both buildings that are occupied (both in general and by type), as well as locations that are NOT occupied, and thus ripe for use.
